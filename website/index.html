ABOUT TONALITE

Tonalite was created to fix the problem of motion. That is, I had my lighting board at the back of the theater, and the lights in front. As the only lighting person, it was getting hard to keep running back and forth. Tonalite solves this issue because it is a web interface that connects to any device with a web browser so you can take it anywhere you want. It has the same feature set as other big-name competetors, and it has architectural presets built-in for permanent installations.
